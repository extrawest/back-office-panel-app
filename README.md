### Installing:

**1) Clone this repo to your folder:**

```
git clone https://github.com/extrawest/back-office-panel-app.git
```

**2. Change current directory to the cloned folder:**

```
cd back-office-panel-app
```

**3) You must have installed Node.js v16.15.0 or higher. Run in terminal in root project directory:**

```
# with npm
npm install --force

# with yarn
yarn install --force

# with pnpm
pnpm install --force
```

**4) Start project. Run in terminal:**

```
# with npm
npm run start:all

# with yarn
yarn run start:all

# with pnpm
pnpm run start:all
```

**5) Test project. Run in terminal:**

```
# with npm
npm run test:all

# with yarn
yarn run test:all

# with pnpm
pnpm run test:all
```
